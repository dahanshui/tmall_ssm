# tmall_ssm  项目介绍                                                                                                                                                                                                                
一个完全模仿天猫的商城网站，进行前后端的交互，支付流程梳理，领会淘宝购物的整个流程。
## 演示地址
* 暂时未提供
## 技术栈
* 后台
   * Spring+SpringMVC+Mybatis+Druid+Mysql
* 前端
   * BootStrap+JQuery+CSS+JavaScript
## 功能
* 基本功能
   * 商品发布、更新
   * 商品购买
   * 权限管理
   * 购物车及支付流程
